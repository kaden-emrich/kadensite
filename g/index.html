Asteroids

Controls: Arrow Keys, W,A,S,D, Space

Out of all of the programming projects that I have done so far, I’d say that Asteroids is probably my favorite. I’ve always liked Atari Asteroids because It’s a great example of a simple concept done well. So, I wanted to take a shot at it.

The graphics in Asteroids are drawn with vectors on an html canvas to stay true to the original game. The game uses no outside engines or libraries. The collision, scoring, movement, etc. is all calculated by code written by me. Good thing I’ve always been good at math!

Many of the mechanics and values in the game went through countless revisions and tweaks. The logic for spawning more asteroids is one such mechanic that I am quite proud of. It feels like a natural progression especially if you don’t pay attention to the values that determine the spawning conditions.

This game took a lot of time and effort so I would appreciate it if you play it. Hope you have fun!
